Installation

To get started, run the following command in your terminal:

npm install babel-fish --save

Or if you prefer yarn, use yarn add babel-fish instead. You can also press Ctrl + C to cancel at any time.

Code Example

Here's how you might use the library in JavaScript:

import { translate } from 'babel-fish';

async function greetAlien(message, language) {
const result = await translate(message, {
  from: 'english',
  to: language,
  politeness: 'maximum'
});

return `Greetings, traveler! ${result}`;
}

// Don't forget to handle Vogons carefully
greetAlien('Hello', 'vogon').catch(console.error);
